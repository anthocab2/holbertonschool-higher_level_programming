# Python - Inheritance

This project focuses on understanding inheritance in Python.

## Topics Covered
- Superclass and subclass
- Inheritance and multiple inheritance
- Attribute and method lookup
- Built-in functions related to inheritance

## Requirements
- Python 3.8.5
- Ubuntu 20.04 LTS
- pycodestyle 2.7.*

## Files
- 0-lookup.py: Returns all available attributes and methods of an object
